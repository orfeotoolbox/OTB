Keyboard and mouse shortcut mappings

Main window
Key Action
CTRL+ O Open file(s)
CTRL+ Q Quit application
Histogram view
Key Action
Left-button horizontal-drag Read histogram values
Middle-button vertical-drag Scale histogram view
Right-button horizontal-drag Shift histogram-view towards lo high
Wheel Scale histogram-view
Image View
(Click in to set focus)
Key Action
CTRL+ P Print image-view (screenshot)
SHIFT+ CTRL+ P Quick Print image-view (quick screenshot)
ALT+ Mouse wheel Adjust effect brush size
SHIFT+ ALT+ Mouse wheel Adjust effect parameter value
Mouse wheel Rotate layers
CTRL+ Mouse wheel Zoom in out
CTRL+ALT+ Mouse wheel Shift dynamics range towards low/high
SHIFT+ CTRL+ALT+ Mouse wheel Scale dynamics range around it's mean value
META+ Mouse wheel Adjust translucency
SHIFT+ META+ Mouse wheel Adjust gamma correction
Key Action
Mouse drag Scroll view
CTRL+ Mouse drag Quick scroll view
↑ Scroll up
↓ Scroll down
← Scroll left
→ Scroll right
CTRL+ ↑ Quick scroll up
CTRL+ ↓ Quick scroll down
CTRL+ ← Quick scroll left
CTRL+ → Quick scroll right
+ Zoom in
- Zoom out
1 Zoom to full resolution (1:1)
2 Zoom to selected layer extent
3 Zoom to all layers extent
Key Action
Delete Delete selected layer
SHIFT+ Delete Delete all layers
Home Select previous layer
End Select next layer
SHIFT+ Home Select first layer
SHIFT+ End Select last layer
Page Up Move layer up within stack
Page Down Move layer down within stack
SHIFT+ Page Up Move layer to top of stack
SHIFT+ Page Down Move layer to bottom of stack
A Apply selected layer settings to All layers
P Use selected layer Projection as reference projection
Q Apply 2% Quantiles to viewed area of selected image layer
SHIFT+ Q Apply 2% Quantiles to full area of selected image layer
Key Action
C Switch to Chessboard effect
D Switch to local Dynamics or local contrast effect
G Switch to Gradient effect
H Switch to Horizontal swipe effect
N Switch to No effect
S Switch to Spectral angle effect
T Switch to local Translucency effect
V Switch to Vertical swipe effect
Layer Stack
Key Action
Delete Delete selected layer
